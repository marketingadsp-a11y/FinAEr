# **App Name**: CrediControl

## Core Features:

- Client Registration: Register clients with basic information, guarantees, and endorsements.
- Loan Creation: Create new loans, specifying the amount and type of plan (e.g., '110 weekly for every 1000 borrowed for 12 weeks').
- Payment History Tracking: Track payment history and balances for each loan.
- Loan Type Definition: Define and manage different types of loans.
- Client Outreach: Use generative AI tool to determine when and how to follow up with clients about their loans or missed payments.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to convey trust and stability. Light blue has been chosen rather than just any arbitrary color because the prompt requests green, and this accent color is closely related, only with a slightly different tint. A more saturated blue would be too jarring.
- Background color: Very light blue (#F0F8FF) for a clean, professional look, derived from the primary color with high brightness and low saturation to fit the light color scheme.
- Accent color: Pale green (#98FB98) to complement the blue and reinforce the financial theme, positioned analogously to the blue on the color wheel and desaturated to not distract from the light color scheme.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable interface.
- Use simple, consistent icons to represent loan statuses and actions.
- Maintain a clean and organized layout for easy navigation and data management.
